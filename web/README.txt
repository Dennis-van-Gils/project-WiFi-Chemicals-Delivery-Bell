HOWTO INSTALL on your web server

1) Adjust the configuration file `\chemicals_delivery\globals.php` to fit
your situation. The fields you should change are:

 '\HASHED_MAC_ADDRESS'
 '\email_tag'
 '\email_from'
 '\email_to'

2) Copy the folder 'chemicals_delivery' to your web server's 'httpdocs' folder.
The webpage should now be found on the internet at the following URL:
https://[your web host]/chemicals_delivery
